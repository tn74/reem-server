Welcome to the REEM Browser

This is a Django based web browser that allows you to view data stored with REEM

  1. Go to the url path "/view/[reem-path]" to see the data stored in Redis
    • For example, if you wanted to see what was stored at "foo.bar.subkey", go to the url "http://127.0.0.1:8000/view/foo.bar.subkey"
  2. Numpy data can be viewed as
    1. A pretty printed list of numbers
    2. An image
    • If you try to something the size of an image array as a list, the server probably won't load because it takes too long generate the html

By default the web server Django connects to is the localhost. If you need to change that, change the REEM_hostname variable at the bottom of Django's configuration variable.

  • The Redis server where the data is stored must be on this machine and bounded to the local interface
